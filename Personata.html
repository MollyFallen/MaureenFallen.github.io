Personata

This was a project done during Startup Weekend. It is a prototype for an app that will allow you to manage your tasks and events automatically by pulling from data that you've already created in your daily life.

Team Members:

Mathias Burton, Alex Trimble, and Imran Peerbhai.

My Contributions:

  • I ideated to brainstorm possible applications of data aggregation.
  • I synthesized the ideas for the project into a preliminary interface.
  • I created a wireframe for Alex to use to create the prototype.
